Flujo completo:
1. Carrusel muestra el mazo de diseños con flechas ← →
2. Toca un diseño → modal con las 4 variantes (garra / ascendente / doble / rocío)
3. Usuaria elige qué pose prefiere → "Quiero este diseño"
4. Si no está logueada → Google login → vuelve con el diseño seleccionado
5. En dashboard → diseño aparece adjunto a la nueva cita → WhatsApp bot lo incluye
